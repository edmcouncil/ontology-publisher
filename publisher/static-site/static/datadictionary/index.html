EDM Council FIBO Glossary

Generated from the OWL, the FIBO glossary is a list of classes and properties defined in FIBO, including their logical definition, curated natural language definition and synonyms where available. As with all of FIBO, there is a Production version of the Glossary, which includes definitions of terms that have been vetted by the FIBO content teams, as well as a Development version, which includes terms that are currently being developed. There are four ways to work with the FIBO Glossary.

  1. Web Searchable
  2. Downloadable as a .csv for working in a spreadsheet
  3. Downloadable as a .xlsx for working in a spreadsheet
  4. Downloadable as RDF annotations for use with FIBO itself
The glossary incudes human-readable versions of the logical relationships in FIBO that are represented in OWL. These definitions are available on their own as annotations on the FIBO classes in rdf-xml and ttl formats.
